Monster Survival is a 2D top-down survival game developed using **WASM-4**.  
The player must survive waves of zombies, collect keys, unlock abilities, and defeat a final boss to win the game.

This project was developed as an academic game programming project.

---

## Gameplay Features

- Zombie survival with multiple waves
- Collectible keys to progress
- Unlockable poison attack
- Final boss battle with platform mechanics
- Health system and invincibility frames
- Pause menu and restart system

## How to Run

1. Install WASM-4:  
   https://wasm4.org

2. Build the game:
```bash
w4 build main.go
```
Run the game:
```bash
w4 run monster-survival.wasm
```
